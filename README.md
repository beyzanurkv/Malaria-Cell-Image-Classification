# Malaria-Cell-Image-Classification

CNN-based model classifies malaria-infected (Parasitized) vs healthy (Uninfected) red blood cells from microscope images using Kaggle dataset.

Features
- CNN Model: 95%+ accuracy (5 epochs)

- Transfer Learning: VGG16 achieves 98%+ validation accuracy (15 epochs)

- Dataset: Cell Images for Detecting Malaria (80/20 train/validation split)

 Quick Start
- git clone <repo-url>
- pip install tensorflow pandas matplotlib numpy
- jupyter notebook malaria-imageclassification.ipynb
